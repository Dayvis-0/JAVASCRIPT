# JAVASCRIPT

Repositorio completo de aprendizaje de JavaScript con material teórico, ejercicios prácticos y proyectos de ejemplo.

## Estructura del Contenido

- **0. Teoria Google**: Fundamentos básicos y conceptos introductorios
- **1. Teoria Soy dalto junior**: Material para nivel principiante
- **2. Teoria Roadmap**: Teoría estructurada por temas específicos
- **2. Practica Roadmap**: Ejercicios prácticos complementarios
- **5. Teoria Soy dalto mid**: Contenido para nivel intermedio
- **Proyectos a duplicar**: Proyectos de ejemplo para practicar

## Temas Cubiertos

Variables y tipos de datos, funciones, objetos, arrays, manipulación del DOM, eventos, programación asíncrona, programación orientada a objetos y conceptos avanzados de JavaScript.
